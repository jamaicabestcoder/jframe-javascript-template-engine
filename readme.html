Overview

Jframe is a lightweight JavaScript templating engine that provides powerful template rendering capabilities with two-way data binding support.

  • Template compilation and rendering
  • Two-way data binding for form elements
  • Conditional rendering with {{#if}}
  • List rendering with {{#each}}
  • HTML escaping for security
  • Complex expressions and method calls
  • DOM element rendering and updates

Installation & Setup

// Import the main instance
import { jframe } from './jframe/index.js';

// Or create your own instance
import { Jframe } from './jframe/index.js';
const engine = new Jframe();

Basic Usage

1. Simple Template Rendering

// Render template string to HTML
const template = '<h1>Hello {{ name }}!</h1>';
const context = { name: 'World' };
const html = jframe.render(template, context);
document.getElementById("app").innerHTML = html;
// Result: '<h1>Hello World!</h1>'

2. DOM Element Rendering

HTML

<div id="app">
  <p>Hello {{ user.name }}!</p>
  <p>You have {{ notifications.length }} notifications</p>
</div>

JavaScript

const element = document.getElementById('app');
const context = {
  user: { name: 'Alice' },
  notifications: [/*...*/]
};

// Render to DOM element
jframe.render(element, context);

Template Syntax

1. Variables

Escaped Output (safe)

<p>{{ title }}</p>
<p>{{ user.profile.name }}</p>

Raw Output (unsafe - use with caution)

<div>{{{ htmlContent }}}</div>

2. Conditionals

{{#if user.isLoggedIn}}
  <p>Welcome back, {{ user.name }}!</p>
{{/if}}

{{#if (eq user.role "admin")}}
  <button>Admin Panel</button>
{{/if}}

Comparison Operators:

Operator Description Example
(eq a b) equals (a === b) (eq user.role "admin")
(neq a b) not equals (a !== b) (neq status "pending")
(gt a b) greater than (a > b) (gt user.age 18)
(gte a b) greater than or equal (a >= b) (gte score 100)
(lt a b) less than (a < b) (lt items.length 10)
(lte a b) less than or equal (a <= b) (lte count max)
(and a b) logical AND (a && b) (and isActive isVerified)
(or a b) logical OR (a || b) (or isAdmin isModerator)

3. Loops

<ul>
  {{#each items}}
    <li>{{ this.name }} - Index: {{ @index }}</li>
  {{/each}}
</ul>

<!-- With complex expressions -->
{{#each users.filter(u => u.active)}}
  <div>{{ this.name.toUpperCase() }}</div>
{{/each}}

Loop Context:

  • this - current item in iteration
  • @index - current index (0-based)

4. Complex Expressions

<!-- Method calls -->
<p>{{ message.toUpperCase() }}</p>

<!-- Arithmetic -->
<p>Total: {{ price * quantity }}</p>

<!-- Array operations -->
<p>Count: {{ items.length }}</p>
<p>First: {{ items[0].name }}</p>

<!-- String operations -->
<p>{{ title.slice(0, 10) }}...</p>

<!-- Arrow functions -->
{{#each items.filter(item => item.price > 100)}}
  <div>{{ this.name }}</div>
{{/each}}

Two-Way Data Binding

Setup

// Enable binding when rendering
const element = document.getElementById('app');
jframe.render(element, context, true); // Third parameter enables binding

Usage in Templates

<form>
  <!-- Text inputs -->
  <input type="text" data-bind="user.name" placeholder="Name">
  
  <!-- Checkboxes -->
  <input type="checkbox" data-bind="user.subscribed"> Subscribe
  
  <!-- Textareas -->
  <textarea data-bind="user.bio"></textarea>
  
  <!-- Select -->
  <select data-bind="user.country">
    <option value="us">US</option>
    <option value="uk">UK</option>
  </select>
</form>

<!-- Display bound values -->
<p>Name: {{ user.name }}</p>
<p>Subscribed: {{ user.subscribed }}</p>

Advanced Features

1. Updating Rendered Elements

// Initial render
const element = document.getElementById('app');
jframe.render(element, initialContext);

// Update with new data
jframe.update(element, {
  user: { name: 'Updated Name' }
});

Best Practices

1. Security

<!-- ✅ Safe - automatically escaped -->
<p>{{ userInput }}</p>

<!-- ❌ Dangerous - only use with trusted content -->
<div>{{{ rawHtml }}}</div>

3. Error Handling

try {
  const html = jframe.render(template, context);
} catch (error) {
  console.error('Template error:', error.message);
}

try {
  jframe.update(element, newContext);
} catch (error) {
  console.error('Update error:', error.message);
}
